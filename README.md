# PennReach

An AI-powered alumni cold outreach tool for Wharton/Penn students.

Upload a LinkedIn profile → AI parses it → find their email → get a personalized draft → send via Gmail.

## Docs

Everything you need is in the `docs/` folder:

| Doc | What it covers |
|-----|---------------|
| [01-PRD.md](docs/01-PRD.md) | Product requirements, user stories, feature specs, priorities |
| [02-ARCHITECTURE.md](docs/02-ARCHITECTURE.md) | Tech stack, database schema, API design, folder structure |
| [03-DEVELOPMENT-GUIDE.md](docs/03-DEVELOPMENT-GUIDE.md) | Git workflow, work split, testing, debugging — written for non-engineers |
| [04-REPLIT-AGENT-PROMPT.md](docs/04-REPLIT-AGENT-PROMPT.md) | Copy-paste prompt to scaffold the entire app with Replit Agent |
| [05-HACKATHON-GAMEPLAN.md](docs/05-HACKATHON-GAMEPLAN.md) | Day-by-day timeline, checklists, demo script, troubleshooting |

## Quick Start

1. Read docs 01-03 to understand the product and architecture
2. Complete the pre-hackathon checklist in doc 05
3. Create a Replit project and paste the prompt from doc 04
4. Follow the hackathon gameplan in doc 05
